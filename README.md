# FlightRisingUserscripts
Some userscripts for Flight Rising.

Most of these are half-finished, but guiImprovements should be fully working. See guiImprovements' changes and features section for a list of just that. When you install guiImprovements, make sure to also install guiImprovementsSandboxed, for a few extra features that needed to be sandboxed for security reasons.
